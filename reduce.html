    ReduceStream is a port of the 'reduce' function to streams: Each chunk emitted is passed through a callback and the result is accumulated, starting with the 'start'initial value.

    If nothing is provided to 'start', the callback will only be called after the first chunk which will be used as the initial accumulator value.

    When the source has no more data to emit, the accumulated value is emitted to the next consumer.
