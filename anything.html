    AnythingStreams are readable streams that can emit anything(including null and undefined values). This is done by hooking into the Readable class internals, which is a hacky and ugly solution, but it gets the job done and the damage is kept contained in this file.

    The class provides an implementation for the '_read' method. Subclasses need to implement the '_next' method to yield more data.
